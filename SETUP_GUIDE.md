# SKY13 Trade Engine Bot - Setup and Installation Guide\n\n## 🚀 Quick Start\n\n### 1. Install Dependencies\n\n```bash\npip install -r requirements.txt\n```\n\n### 2. Configure IIFL Credentials\n\n**Copy the example configuration:**\n```bash\ncp .env.example .env\n```\n\n**Edit `.env` with your IIFL credentials:**\n```bash\n# IIFL Broker Credentials\nIIFL_USERNAME=your_client_code        # Your IIFL client code\nIIFL_PASSWORD=your_password            # Your IIFL password\nIIFL_API_KEY=your_api_key              # Optional API key\nIIFL_API_SECRET=your_api_secret        # Optional API secret\n\n# Telegram Bot (Optional)\nTELEGRAM_BOT_TOKEN=your_bot_token      # Get from @BotFather on Telegram\nTELEGRAM_CHAT_ID=your_chat_id          # Your Telegram chat ID\n\n# Trading Settings\nTRADING_MODE=paper                     # Start with 'paper' for testing\nSYMBOL=NIFTY                           # Trading symbol\nTIMEFRAME=5                            # Candle timeframe in minutes\nFAST_EMA=9                             # Fast EMA period\nSLOW_EMA=21                            # Slow EMA period\nDAILY_LOSS_LIMIT=3000                  # Daily loss limit in rupees\nQUANTITY=1                             # Lot size to trade\n```\n\n### 3. Get IIFL API Credentials\n\n1. **Register on IIFL Developer Portal:**\n   - Go to https://developers.iiflcapital.com/\n   - Sign up and get API credentials\n\n2. **Generate API Keys:**\n   - Log in to your IIFL account\n   - Navigate to API settings\n   - Generate USER_KEY and ENCRYPTION_KEY\n\n3. **Add to `.env`:**\n   ```bash\n   IIFL_USERNAME=your_client_code\n   IIFL_PASSWORD=your_password\n   ```\n\n### 4. Set Up Telegram Alerts (Optional)\n\n1. **Create a Telegram Bot:**\n   - Message @BotFather on Telegram\n   - Follow the instructions to create a new bot\n   - Copy the bot token\n\n2. **Get Your Chat ID:**\n   - Start a chat with your new bot\n   - Go to: `https://api.telegram.org/bot<YOUR_TOKEN>/getUpdates`\n   - Find your chat ID from the response\n\n3. **Add to `.env`:**\n   ```bash\n   TELEGRAM_BOT_TOKEN=your_bot_token\n   TELEGRAM_CHAT_ID=your_chat_id\n   ```\n\n### 5. Run the Bot\n\n**Paper Trading Mode (Recommended for testing):**\n```bash\npython main.py\n```\n\n**Live Trading Mode (When ready):**\n```bash\n# Change TRADING_MODE=live in .env\npython main.py\n```\n\n---\n\n## 📋 Configuration Guide\n\n### Trading Symbols\n\n**Supported Indices:**\n- NIFTY (default)\n- BANKNIFTY\n- FINNIFTY\n- MIDCPNIFTY\n- SENSEX\n\n### Strategy Parameters\n\n| Parameter | Default | Range | Description |\n|-----------|---------|-------|-------------|\n| FAST_EMA | 9 | 5-20 | Fast EMA period |\n| SLOW_EMA | 21 | 20-50 | Slow EMA period |\n| RSI_PERIOD | 14 | 10-20 | RSI calculation period |\n| RSI_OVERBOUGHT | 70 | 60-80 | RSI overbought threshold |\n| RSI_OVERSOLD | 30 | 20-40 | RSI oversold threshold |\n| ATR_PERIOD | 14 | 10-20 | ATR calculation period |\n| ADX_PERIOD | 14 | 10-20 | ADX calculation period |\n\n### Risk Management Settings\n\n```\nDAILY_LOSS_LIMIT=3000         #
